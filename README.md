## Run Locally

Clone the project

```bash
git clone https://github.com/zz0-0/dial_treeview
```

Go to the project directory

```bash
cd dial_treeview
```

Install dependencies

```bash
flutter pub get
```

Run the project

- Option 1: VSCode

```bash
cd example/lib
# click Run in Run|Debug|Profile
```

- Option 2: Flutter CLI

```bash
cd example/lib
flutter run
# pick the platform you want to run with
```
